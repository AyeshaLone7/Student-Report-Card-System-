Student Report Card System

Description

This project is a simple Student Report Card System developed to manage student marks and generate report cards.
It allows users to enter student information and calculate grades based on marks.

Features

- Add student name and marks
- Calculate total marks
- Generate grades
- Display student report card
- Edit marks

Technology Used

- (c++)

Purpose

This project was created as a university assignment to practice programming concepts and basic data management.
